# 🖥️ Python GUI Development

## 📌 Overview

This repository contains Python Graphical User Interface (GUI) concepts and projects using **Tkinter**, Python's built-in GUI library. It is designed for beginners and intermediate learners who want to build desktop applications with interactive user interfaces.

---

## 📚 Topics Covered

* Introduction to GUI Programming
* Tkinter Basics
* Creating Windows
* Labels
* Buttons
* Entry Widgets
* Text Widget
* Frames
* Canvas
* Checkbuttons
* Radiobuttons
* Listbox
* Combobox
* Menu Bar
* Message Box
* File Dialog
* Color Chooser
* Images in Tkinter
* Event Handling
* Layout Managers

  * Pack
  * Grid
  * Place
* Object-Oriented Tkinter
* Multiple Windows
* Themes (ttk)
* Basic GUI Design Principles

## ▶️ Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/Python-GUI.git
```

### Navigate to the project

```bash
cd Python-GUI
```

### Run a project

```bash
python filename.py
```

## 📦 Requirements

Install any required libraries (if used):

```bash
pip install pillow
pip install qrcode
```

> **Note:** Tkinter comes pre-installed with most Python distributions.
